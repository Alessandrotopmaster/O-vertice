# 🚀 VÉRTICE - Plataforma Completa

## Sandro Rhilmanelly

### 📦 Sobre o Projeto

**VÉRTICE** é uma plataforma completa que reúne em um só lugar:

- 🏪 **Criador de Lojas de Afiliados** - Importação automática de Mercado Livre, Shopee, Amazon
- 📚 **Criador de E-books** - Modelos prontos ou criação livre com PDF profissional
- 🌐 **Sites e Fanpages** - Modelos testados para alta conversão
- 📱 **Divulgação em Redes** - Facebook, Instagram, YouTube, TikTok
- 📊 **Tráfego Pago** - Google Ads, Meta Ads, TikTok Ads simplificados
- 💲 **Planos e Faturamento** - Grátis • Básico • Avançado • Premium

### 🎯 Diferenciais

- ✅ **Regra de Ouro:** Chaves e configurações SEMPRE no .env
- ✅ **Duas formas de usar:** Modelo pronto OU Criar do zero
- ✅ **Interface tão simples:** Qualquer pessoa usa
- ✅ **Renda dupla:** Assinaturas + Comissões automáticas
- ✅ **Firebase integrado:** Auth, Firestore, Storage
- ✅ **PWA:** Funciona como app no celular
- ✅ **Offline:** Funciona sem internet

### ⚡ Como Instalar

```bash
# 1. Clone o repositório
git clone https://github.com/SEU-USUARIO/vertice-platform.git

# 2. Instale as dependências
npm install

# 3. Configure as variáveis de ambiente
cp .env.example .env
# Edite o .env com suas credenciais do Firebase

# 4. Execute o servidor
npm start
